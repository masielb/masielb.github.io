Project Takeoff

Links

Deployed Version: https://masielb.github.io/project-takeoff/

GitHub Repo: github.com/masielb/project-takeoff

Concept

  • Description: Project Takeoff will help users plan trips to Mars!
  • Motivation: With SpaceX driving innovation in space transportation services and leading the conversation in colonization of Mars, it is only a matter of time before travel to Mars becomes a critical industry.
  • User story: AS A fictional visitor to Mars I WANT to see the Martian landscape from multiple points of view SO THAT I can see what Mars looks like up close.

Process

  • Technologies Used: HTML/CSS/JS, jQuery, moment.js, Foundation Library, NASA Public APIs
  • Tasks & Roles: Toks & Masiel took on front-end UI design and development; Max & Grant took on back-end development.
  • Challenges & Successes: Valid user date input for image search. Designing an app that organizes complex data in a user-friendly and approachable way.
